# 2026-03-14

## 股票分析專案

### 改用 LLM 做意圖路由

**問題：** 硬編碼規則維護困難，用戶說法稍有變化就無法識別

**解決方案：** LLM 負責路由（判斷意圖 + 調用工具），Python 負責執行

**架構：**
```
用戶輸入 → LLM (判斷意圖 + 調用工具) → Python (驗證 + 執行) → 結果
```

**協定：**
- LLM 只調用定義好的工具
- Python 驗證 tool_name 後執行

**新增檔案：**
- `llm_tool_schemas.py` - OpenAI function calling 格式
- `test_intent_router.py` - 本地測試腳本

**修改檔案：**
- `intent_parser.py` - 改用 LLM tool calling
- `llm_system_prompt.txt` - 教 LLM 判斷意圖
- `llm_tools.json` - 工具定義

**測試：**
- 大盤/總經查詢正確路由
- 股票查詢正確調用 search_stock
- 「幫助」「推薦」由 preprocess 快速處理
